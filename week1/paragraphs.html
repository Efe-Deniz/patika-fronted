PARAGRAPHS

The pre Attribute

The pre attribute is used to specify that the text inside the element should be displayed in a fixed-width font and that whitespace should be preserved. 

    This    is    preformatted
    text with    preserved whitespace.

If we had done it with the p tag

This is preformatted text preserved whitespace.

The br Attribute

The br attribute is used to insert a line break in the text. It is an empty element, meaning it does not have a closing tag.

This is the first line.
This is the second line.
This is the third line.
