HTML Attributes

HTML Global attributes

The class Attribute

The class attribute is used to specify one or more class names for an HTML element. Classes can be used by CSS and JavaScript to select and access specific elements.

Example

The contenteditable Attribute

The contenteditable attribute is used to specify whether the content of an element is editable or not. When set to "true", the content of the element can be edited by the user.

This content is editable. Click here to edit.

The data-* Attribute

The data-* attribute is used to store custom data private to the page or application. The data can be accessed using JavaScript.

This div has custom data attributes.

The dir Attribute

The dir attribute is used to specify the text direction for the content in an element. It can take the values "ltr" (left to right), "rtl" (right to left), or "auto".

The id Attribute

The id attribute is used to specify a unique identifier for an HTML element. The value of the id attribute must be unique within the HTML document.

This paragraph has a unique ID.

The lang Attribute

The lang attribute is used to specify the language of the content in an element. This can help search engines and browsers to better understand the content.

This paragraph is in English.

Este párrafo está en español.

The title Attribute

The title attribute is used to provide additional information about an element. This information is often displayed as a tooltip when the user hovers over the element.

Hover over this paragraph to see the tooltip.

The translate Attribute

The translate attribute is used to specify whether the content of an element should be translated or not. It can take the values "yes" or "no".

This content should not be translated.

The img Attribute

The src attribute is used to specify the URL of the image to be displayed.

The alt attribute is used to provide alternative text for the image if it cannot be displayed.

The a Attribute

The href attribute is used to specify the URL of the page the link goes to.

Visit Example.com

The video Attribute

The controls attribute is used to specify that video controls should be displayed (such as a play/pause button, volume control, etc.).

The hidden Attribute

The hidden attribute is used to hide an element from the page. The element will not be displayed, but it will still be present in the HTML code.

This paragraph is hidden and will not be displayed on the page.

The tabindex Attribute

The tabindex attribute is used to specify the tab order of an element when the user navigates through the page using the Tab key.

The draggable Attribute

The draggable attribute is used to specify whether an element is draggable or not. When set to "true", the element can be dragged using the mouse.

Drag me!

The spellcheck Attribute

The spellcheck attribute is used to specify whether the content of an element should be checked for spelling errors or not. It can take the values "true" or "false".

The accesskey Attribute

The accesskey attribute is used to specify a shortcut key to activate or focus an element.

The target Attribute

The target attribute is used to specify where to open the linked document. Common values are "_blank" (open in a new tab/window) and "_self" (open in the same frame).

Wikipedia.org

The rel Attribute

The rel attribute is used to specify the relationship between the current document and the linked document. Common values include "noopener", "noreferrer", and "nofollow".

Example.com with rel attributes

The download Attribute

The download attribute is used to specify that the target link should be downloaded when clicked, rather than navigating to it.

Download Dummy PDF

The loading Attribute

The loading attribute is used to specify whether a resource should be loaded lazily or eagerly. Common values are "lazy" (load when needed) and "eager" (load immediately).

The decoding Attribute

The decoding attribute is used to specify how the browser should decode the image. Common values are "async" (decode asynchronously) and "sync" (decode synchronously).
